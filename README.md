- [队列](https://zh.wikipedia.org/wiki/%E9%98%9F%E5%88%97)
```
 队列，又称为伫列（queue），是先进先出（FIFO, First-In-First-Out）的线性表。
在具体应用中通常用链表或者数组来实现。队列只允许在后端（称为rear）进行插入操作，在前端（称为front）进行删除操作。
队列的操作方式和堆栈类似，唯一的区别在于队列只允许新数据在后端进行添加。
```
